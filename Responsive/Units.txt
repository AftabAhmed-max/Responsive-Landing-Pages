Units :
1. px - according to the pixels of the screen
2. % - According to the percentage of the parents
3. vh - according to the viewport height
4. vw - according to the viewport width
5. vmax - if height is max it takes reference of height and if width is max it will take refernce of width
6. vmin - if height is min it takes reference of height and if width is min it will take refernce of width
7. em - give a size to the parent (eg:10px) and in child if you give 1em so that means 1em = 10px
7. rem - give a size to the html (eg:10px) and in any elements if you give 1rem so that means 1rem = 10px

CSS Media Queries:
1. max-width and max-height - eg: max-width 600px means 600px se kam and zyada se zyada 600px usse zyada nahi.
2. min-width and min-height - eg: min-width 768px 768px se zyada and kam se kam 768px usse kam nahi
 

Things to keep in mind:
